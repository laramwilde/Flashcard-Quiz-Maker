# FLASHCARD QUIZ MAKER
Program that allows the user to create flashcard and quiz themselves on it. Includes file handling to help me practice it.

## Features
- Load flashcards from a file
- Save flashcards to a file
- Add new flashcards
    - Includes term and definition
- List all flashcards
- Quiz mode
    - Includes score tracking in quiz mode
- Allows deleting cards
- Allows editing cards
- Quit program feature

## Why I made this
I'm currently doing a T-Level in college in Digital Production, I thought it would be useful to make a mini flashcard program I could use. (Although I probably wont.) This project  will help me work on file handling & dictionaries.
